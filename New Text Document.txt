Web diễn đàn trao đổi code : 
Người dùng : 
- Đăng ký
- Đăng nhập
- Xem thông tin các khóa học
- Xem chi tiết khóa học
- Đăng bài viết (câu hỏi hoặc code )
- Bình luận bài viết
- Trả lời bình luận
- Xem các bài viết người dùng khác đăng
Admin :
- Quản lý người dùng : hiển thị danh sách, chặn người dùng 
- Quản lý bài viết : duyệt bài, chặn bài
- Quản lý thông tin khóa học : hiển thị danh sách, thêm, sửa, xóa
